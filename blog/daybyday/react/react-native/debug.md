## 开发者菜单

### 开启调试快捷键(iOS)

确保模拟器中 `Simulator` => `Hardware` => `Keyboard` => `Connect Hardware Keyboard` 处于 ☑️ 状态.

### 访问 APP 内开发菜单

- 真机调试
  - 摇晃设备
  - `Simulator` => `Hardware` => `Shake Gesture`
- 模拟器
  - `iOS` `Commond ⌘` + `D`
  - `Android` `Commond ⌘` + `M`
